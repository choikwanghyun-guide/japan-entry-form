📋 영수증 JSON 붙여넣기 — Claude 채팅에서 받은 JSON을 정산서에 자동 적용

📚 사용법 보기 (Claude에 보낼 프롬프트 포함)
1. Claude 채팅에 영수증 사진·스캔(JPG/PNG/PDF) 업로드
2. 아래 프롬프트를 복사해서 함께 전송: 
이 영수증들을 가이드 정산서용 JSON으로 변환해줘.

스키마:
- hotel/lunch/dinner/tour: 배열. 각 항목은 {date:"YYYY-MM-DD", name, condition?, people?, unitPrice?, amount}
- bus/guide/etcYen: 항목명을 key로 한 객체. 값은 {amount, unitPrice?, note?}
- etc: 원화 기타경비. 항목명을 key로 한 객체. 값은 {amount}

카테고리 분류 기준 (영수증 위의 수기 메모 또는 내용으로 판단):
- hotel: 호텔비
- lunch/dinner: 중식/석식
- tour: 관광지 입장료
- bus.{기사식대,오차다이,버스비,도로비,주차대,회송대,기사 숙박,...}
- guide.{식대,통신료,출장비,교통비}
- etcYen.{출국세,부두세,...} (엔화 기타)
- etc.{...} (원화 기타)

다른 설명 없이 JSON만 응답해줘. ```json 코드 블록은 사용 OK.


3. Claude가 응답한 JSON을 통째로 복사 → 아래 칸에 붙여넣기 → "🔄 적용" 버튼

예시 JSON: 
{
  "hotel": [{"date":"2026-05-24","name":"SOAR RESORT","condition":"조식·입탕 포함","amount":195050}],
  "lunch": [{"date":"2026-05-25","name":"OUTDOOR PARTNERS","people":"26","unitPrice":"1200","amount":31200}],
  "dinner": [{"date":"2026-05-24","name":"焼肉研究所","amount":31200}],
  "bus": {"기사식대":{"amount":1500,"note":"5/24 夕食代"}, "오차다이":{"amount":4000}},
  "etcYen": {"출국세":{"amount":27000}, "부두세":{"amount":12500}}
}

📎 영수증 첨부 (스캔/사진)

🤖 AI 자동 분석 설정 (선택 사항)
🔑 Gemini 키 받는 법: Google AI Studio(aistudio.google.com) 접속 → 구글 로그인 → 좌측 "Get API key / API 키 만들기" → 키 복사해서 위 칸에 붙여넣기 (무료 사용량 있음).
⚠️ API 키는 이 브라우저(localStorage)에만 저장됩니다. HTML 파일을 남과 공유하면 키도 함께 넘어가니, 가이드마다 본인 키를 각자 넣게 하세요. 빈 칸으로 두면 자동 분석이 즉시 꺼집니다.
API 키 미설정 — 자동 분석 비활성

JPG, PNG, PDF 파일 지원 · 여러 장 한꺼번에 가능 · 이미지는 자동 압축됩니다
💾 정산서 저장 / 불러오기
현재 정산서 전체(입력·표·첨부 영수증)를 파일 1개로 저장하거나, 저장한 파일을 불러옵니다. 단체별로 보관·백업·공유할 수 있어요.
📤 랜드사 양식으로 내보내기
현재 입력된 데이터를 각 랜드사 정산서 양식에 맞게 자동으로 채워 새 엑셀 파일로 저장합니다. 데이터가 없는 칸은 빨간색 "확인"으로 표시됩니다.
⚠ 전체 입력 항목 삭제
새 정산서 작성을 시작할 때 사용합니다. 모든 입력값과 첨부 영수증이 초기화됩니다.
⚠️ 인원 표기 방법 — 숫자만 적으면 손님만, 수식이면 자동으로 합산해서 단가를 계산합니다.
예) 25 = 손님 25명 / 25+1 = 손님 25 + 가이드 1 (= 26명) / 25+2 = 손님 25 + 가이드 1 + 기사 1 (= 27명)
※ 중식·석식의 단가는 (지불금액 ÷ 인원)으로 자동 계산됩니다. 인원 칸을 채우면 단가가 자동으로 표시돼요.
